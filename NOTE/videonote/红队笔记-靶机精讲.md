# 红队笔记-靶机精讲

## 信息收集
常规渗透测试：端口扫描，详细信息扫描，UDP扫描，漏洞脚本扫描  

可以通过自己对端口的熟悉，按需扫描，即使不是常规端口，也可以发现后再调整，在一定程度上减少暴露，更像用户，增加数据的灰度



### 主机发现

sudo nmap -sn 自己的网段


## other
base64解码命令:  
echo "字符串" | base64 -d

若解出来还是base64，可以这样：
echo "字符串" | base64 -d | base64 -d
